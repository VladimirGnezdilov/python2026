# Домашнее задание. Урок 45. Работа с файлами

## Общие требования

- Работай с текстовыми файлами в кодировке UTF-8.
- Пути к файлам храни в переменных (например, `path_in`, `path_out`).
- Проверяй существование файла через `os.path.exists`.
- Не используй сторонние библиотеки.

---

## Задача 1. Список участников из файла

Дан путь к файлу `participants.txt`. В файле, возможно, есть пустые строки и пробелы.

### Требуется

Напиши функцию `print_participants(path)`, которая:

- Если файла по пути `path` не существует, выводит:

```
Нет файла со списком участников
```

- Если файл существует, но в нём нет ни одного имени (файл пустой или содержит только пустые строки и пробелы), выводит:

```
Список участников пуст
```

- Если имена есть, выводит список в формате:

```
Участники (N):
1) <имя>
2) <имя>
...
```

### Примечания

- Именем считается строка после `strip()`.
- Порядок имён сохраняй как в файле.

## Задача 2. Добавление новой записи в журнал

Есть файл `log.txt`. Каждая строка, это одна запись.

### Требуется

1. Запросить у пользователя строку `message`.
2. Если `message` после `strip()` пустая, вывести:

```
Пустое сообщение. Ничего не записано
```

3. Иначе дописать строку в конец файла `log.txt` (режим `a`).
4. После записи вывести:

```
Запись добавлена
```

### Уточнение

- Если файла `log.txt` не существует, он должен создаться автоматически.

## Задача 3. Отчёт по оценкам

Есть файл `scores.txt`. В нём строки вида:

```
<имя> <оценка>
```

Примеры:

```
Ivan 5
Olga 4
Petr 3
```

### Требуется

Напиши программу, которая:

1. Если файла нет, выводит:

```
Нет файла с оценками
```

2. Иначе читает все корректные строки.
   - Корректная строка: ровно 2 части, вторая часть это целое число.
   - Некорректные строки пропускай.
3. Выводит в консоль:

```
Учеников: N
Средний балл: X
Максимум: <имя> <оценка>
Минимум: <имя> <оценка>
```

4. И сохраняет тот же отчёт в файл `report.txt` (режим `w`).

### Примечания

- `X` можно выводить как число с 2 знаками после запятой.
- Если после фильтрации не осталось ни одной корректной строки, вывести:

```
Нет корректных данных
```

и `report.txt` не создавать.
