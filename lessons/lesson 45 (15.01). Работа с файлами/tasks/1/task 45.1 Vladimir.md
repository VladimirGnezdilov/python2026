# Задание 45.1

Напиши функцию `print_attendees(path: str)`, которая выводит на экран список участников, записанных в текстовом файле.

## Формат входного файла

- В файле может быть несколько строк.
- Каждая строка может содержать имя участника.
- В строках могут встречаться лишние пробелы в начале и в конце.
- В файле могут встречаться пустые строки.

Пример содержимого файла:

```
  Анна Иванова

Пётр Смирнов  
  Мария
```

## Требования к функции

- Если файла по пути `path` не существует, вывести:

```
Нет файла со списком участников
```

- Если файл существует, но в нём нет ни одного имени (например, файл пустой или содержит только пустые строки и пробелы), вывести:

```
Список участников пуст
```

- Если имена есть, вывести список в таком формате:

```
Участники (N):
1) <имя>
2) <имя>
...
```

Где `N` это количество участников после удаления пробелов и пустых строк.
