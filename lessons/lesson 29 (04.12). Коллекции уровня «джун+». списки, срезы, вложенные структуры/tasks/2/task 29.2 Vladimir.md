# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Отрабатываем выбор нужных частей списка без дополнительных циклов.

## Условие задачи
Ты фиксируешь интервалы обслуживания автомобиля. Нужно:
1. Показать первые три интервала для проверки у мастера.
2. Получить последние четыре интервала, чтобы оценить актуальные работы.
3. Сформировать срез, который берет интервалы с индекса 1 по 6.
4. Создать срез с шагом 3, чтобы увидеть только крупные ТО.

## Пример входных данных
```python
service = ["масло 10 тыс", "фильтры 20 тыс", "тормоза 30 тыс", "свечи 40 тыс", "антифриз 50 тыс", "ходовая 60 тыс", "ГУР 70 тыс", "кузов 80 тыс"]
```

## Пример выходных данных
```
Начало истории: ['масло 10 тыс', 'фильтры 20 тыс', 'тормоза 30 тыс']
Свежие работы: ['свечи 40 тыс', 'антифриз 50 тыс', 'ходовая 60 тыс', 'ГУР 70 тыс', 'кузов 80 тыс']
Средний блок: ['фильтры 20 тыс', 'тормоза 30 тыс', 'свечи 40 тыс', 'антифриз 50 тыс', 'ходовая 60 тыс']
Ключевые ТО: ['масло 10 тыс', 'свечи 40 тыс', 'ГУР 70 тыс']
```

## Критерии успешного выполнения
- Все выборки построены срезами
- Используются и положительные, и отрицательные индексы
- Каждый вывод сопровождается подписью
- Исходный список сохраняется без изменений

## Подсказки
- `[:3]` быстро берет первые элементы
- Хвост через `[-4:]` покажет последние записи
- Диапазон `[1:6]` вернет элементы с индекса 1 по 5
- Шаг `3` поможет выделить редкие события

## Частые ошибки и ограничения
- Не ставь шаг 0
- Следи, чтобы границы не выходили за пределы списка
- Не путай включаемую и исключаемую границы
- Не изменяй исходный список, все манипуляции через `[:]`
