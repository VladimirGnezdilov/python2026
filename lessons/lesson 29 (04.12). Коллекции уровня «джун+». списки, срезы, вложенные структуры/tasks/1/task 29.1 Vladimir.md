# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Учимся хранить упорядоченные данные, работать с индексами и проходить по списку циклом.

## Условие задачи
Собери список инструментов для домашней мастерской. Создай базовый перечень, добавь покупки из интернет-корзины в конец, выведи общее количество, покажи второй и последний элементы. Заверши вывод нумерованным списком, чтобы проверить, всё ли есть в гараже.

## Пример входных данных
```python
starter_tools = ["отвёртка", "молоток", "уровень"]
cart = ["шуруповёрт", "набор бит"]
```

## Пример выходных данных
```
Всего предметов: 5
Второй предмет: молоток
Последний предмет: набор бит
1. отвёртка
2. молоток
3. уровень
4. шуруповёрт
5. набор бит
```

## Критерии успешного выполнения
- Базовый список создан и расширен методами списка
- Количество элементов рассчитано через `len`
- Индексы используются для доступа к внутренним и последним элементам
- Итоговый список выведен построчно с нумерацией

## Подсказки
- Сначала добавь новые инструменты через `extend`, чтобы порядок сохранился
- Используй индексы `1` и `-1` для нужных элементов
- `enumerate(..., start=1)` избавит от ручного счётчика

## Частые ошибки и ограничения
- Не обращайся к индексу `1`, если список короче двух элементов
- Избегай повторного добавления одного и того же инструмента
- Не выводи список одной строкой, проходись циклом
- Убедись, что все операции выполнены до финального вывода
