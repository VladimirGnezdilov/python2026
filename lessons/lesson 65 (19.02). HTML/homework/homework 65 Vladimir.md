# Домашнее задание 65

## Задание 1: Улучшение страницы-визитки (обязательно)

На основе материалов занятия создай или доработай свою страницу-визитку.

### Требования к HTML:

1. **Семантическая структура:**
   - Используй теги `header`, `main`, `footer`, `section`, `article`
   - Один `h1` на страницу
   - Логичная иерархия заголовков `h2` → `h3`

2. **Контент:**
   - Шапка с твоим именем и краткой информацией
   - Минимум 3 секции в основной части:
     - Обо мне
     - Навыки (список из 5+ пунктов)
     - Проекты (минимум 2 карточки)
   - Подвал с контактами (минимум 2 ссылки)

3. **Изображения и ссылки:**
   - Аватар в шапке с атрибутом `alt`
   - Все ссылки имеют атрибут `href`
   - Минимум одна ссылка с `mailto:`

### Требования к CSS:

1. **Базовые стили:**
   - Подключение через `link` в `head`
   - `box-sizing: border-box` для всех элементов
   - Базовые стили для `body` (шрифт, цвет фона)

2. **Селекторы и классы:**
   - Используй классы для стилизации (не только теги)
   - Примени разные типы селекторов (по тегу, по классу, комбинированные)
   - Используй хотя бы один псевдокласс (`:hover`)

3. **Box Model и отступы:**
   - Согласованные отступы между секциями
   - Внутренние отступы в карточках
   - Используй `border-radius` для скругления

4. **Flexbox:**
   - Выровняй элементы в шапке через flex
   - Сделай сетку проектов через flex с переносом

5. **Адаптивность:**
   - Страница не должна иметь горизонтального скролла на узком экране
   - Изображения должны адаптироваться под ширину контейнера

## Задание 2: Дополнительные элементы (на выбор)

Выбери минимум 2 пункта для реализации:

### Вариант A: Улучшенная типографика
- Добавь `line-height` для лучшей читаемости
- Стилизуй первую букву параграфа (`::first-letter`)
- Сделай разные размеры для заголовков

### Вариант B: Интерактивные элементы
- Добавь `:hover` эффекты для всех ссылок
- Сделай плавные переходы (`transition`)
- Добавь `:focus` стили для интерактивных элементов

### Вариант C: Декоративные элементы
- Используй псевдоэлементы `::before` или `::after`
- Добавь иконки или декоративные линии
- Сделай градиентный фон для одного из блоков

### Вариант D: Расширенная структура
- Добавь секцию "Образование" или "Опыт работы"
- Создай более сложную структуру карточек проектов
- Добавь боковую панель (sidebar)
