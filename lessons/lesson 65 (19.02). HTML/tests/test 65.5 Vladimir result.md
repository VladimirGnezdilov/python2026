# Вопрос 1. Блок 6. Flexbox: выравнивание и простая сетка
  
### Содержание и вопросы

- Что такое flex-контейнер и flex-элемент?
- Чем отличаются `justify-content` и `align-items`?
- Что делает `gap`?
- Как сделать перенос карточек?

### Материал (лаконично)

- Flexbox это способ разложить элементы в ряд или в колонку и управлять выравниванием.
- Когда ты включаешь `display: flex`, дети внутри становятся flex-элементами.
- `justify-content` это распределение по главной оси. Обычно это горизонталь.
- `align-items` это выравнивание по другой оси. Обычно это вертикаль.
- `gap` это расстояние между элементами, без “костылей” с `margin`.
- `flex-wrap` позволяет элементам переноситься, если места не хватает.

   - элементы были аккуратно выровнены

5. Для секции проектов:
   - сделай контейнер карточек flex-контейнером

   - добавь промежутки

Ограничения:

- Сетка должна адекватно реагировать на сужение окна.

### Псевдокод решения

```text
IN CSS_FILE
  MAKE HEADER_ROW_LAYOUT
    SET DISPLAY_FLEX
    SET ALIGNMENT
    SET GAP

  MAKE CARDS_GRID
    SET DISPLAY_FLEX
    ENABLE WRAP
    SET GAP

  FOR EACH CARD
    SET FLEX_BASIS_OR_WIDTH_RULE
END
```
