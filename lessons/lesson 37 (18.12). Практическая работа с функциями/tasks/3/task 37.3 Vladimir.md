# Урок 37. Блок 3. Самостоятельное задание

1. Напиши функцию `push_score(score, scores=None)`.
   - Добавляет `score` в список `scores`.
   - Возвращает список.
2. Проверь 3 сценария.
   - Вызов без передачи `scores` два раза подряд.
   - Вызов с передачей уже существующего списка `scores`.
3. Дополнительное усложнение.
   - Напиши функцию `top_scores(scores, n=3)`.
   - Возвращает список из `n` самых больших значений.
