# Урок 37. Блок 4. Самостоятельное задание

1. Напиши функцию `make_multiplier(factor)`.
   - Она возвращает внутреннюю функцию `mul(x)`.
   - `mul(x)` возвращает `x * factor`.
2. Создай множители.
   - `double = make_multiplier(2)`.
   - `triple = make_multiplier(3)`.
   - Проверь `double(10)` и `triple(10)`.
3. Напиши функцию `mean(*args)`.
   - Если аргументов нет, вернуть `None`.
   - Иначе вернуть среднее арифметическое.
4. Напиши функцию `build_profile(**kwargs)`.
   - Достает `name` через `kwargs.get("name", "Без имени")`.
   - Достает `age` через `kwargs.get("age", 0)`.
   - Возвращает словарь профиля.
