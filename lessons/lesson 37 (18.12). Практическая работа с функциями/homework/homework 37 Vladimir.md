# Домашнее задание. Урок 37 (18.12). Практическая работа с функциями

## Формат сдачи
1. Создай файл `homework_37.py`.
2. Внутри файла реализуй функции из заданий ниже.
3. В конце файла добавь несколько тестовых вызовов через `print(...)`, чтобы было видно, что функции работают.
4. Загрузите в журнал ссылку на файл или репозиторий с решением.

## Требования
- Используй функции и `return`, а не только `print` внутри функций.
- Следи за параметрами. Где нужно, используй значения по умолчанию.
- Не используй глобальные переменные для хранения результата между вызовами.

## Задания

## 1) Базовые функции и `return`
1. Напиши функцию `rectangle_info(width, height)`.
   - Она должна возвращать кортеж из двух значений. Площадь и периметр.
2. Напиши функцию `is_adult(age)`.
   - Возвращает `True`, если возраст 18 или больше.
   - Иначе возвращает `False`.
3. Напиши функцию `safe_div(a, b)`.
   - Если `b == 0`, вернуть `None`.
   - Иначе вернуть результат деления `a / b`.

## 2) Позиционные и именованные аргументы
1. Напиши функцию `delivery_price(city, weight_kg, urgent)`.
   - Если `urgent=True`, формула `500 + 30 * weight_kg`.
   - Иначе формула `300 + 20 * weight_kg`.
   - `city` пока используется только для читаемости и будущих условий.
2. Вызови `delivery_price` 4 раза.
   - Два раза позиционно.
   - Два раза именованно.
3. Сделай один намеренно неправильный вызов в виде комментария, чтобы увидеть `TypeError` или `SyntaxError` и понять причину.

## 3) Значения по умолчанию. `None` pattern
1. Напиши функцию `push_score(score, scores=None)`.
   - Добавляет `score` в список `scores`.
   - Возвращает список.
2. Проверь 3 сценария.
   - Вызов без передачи `scores` два раза подряд.
   - Вызов с передачей уже существующего списка `scores`.
3. Дополнительное усложнение.
   - Напиши функцию `top_scores(scores, n=3)`.
   - Возвращает список из `n` самых больших значений.

## 4) `*args`, `**kwargs`, распаковка. Замыкание
1. Напиши функцию `make_multiplier(factor)`.
   - Она возвращает внутреннюю функцию `mul(x)`.
   - `mul(x)` возвращает `x * factor`.
2. Создай множители.
   - `double = make_multiplier(2)`.
   - `triple = make_multiplier(3)`.
   - Проверь `double(10)` и `triple(10)`.
3. Напиши функцию `mean(*args)`.
   - Если аргументов нет, вернуть `None`.
   - Иначе вернуть среднее арифметическое.
4. Напиши функцию `build_profile(**kwargs)`.
   - Достает `name` через `kwargs.get("name", "Без имени")`.
   - Достает `age` через `kwargs.get("age", 0)`.
   - Возвращает словарь профиля.

## Критерии проверки
- Код запускается без ошибок.
- Каждая функция соответствует описанию и корректно возвращает результат.
- `push_score` не копит значения между вызовами, если `scores` не передан.
- В `mean` нет деления на ноль при вызове без аргументов.
