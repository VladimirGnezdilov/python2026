# Задание 4. Условные конструкции и циклы в Python

### Описание
Напиши программу. игра угадай число.
Компьютер загадывает целое число от 10 до 50 включительно (можно просто сохранить его в переменной).
Пользователь пытается угадать число. вводя варианты в цикле.

Правила.
- если число меньше загаданного. вывести `Мало`
- если число больше загаданного. вывести `Много`
- если число равно загаданному. вывести `Угадано` и завершить цикл
- если пользователь вводит 0. программа завершается с сообщением `Выход по желанию пользователя`

Реализуй логику с помощью цикла while.

### Пример входных данных
```
Попытки: 12 3 25 0
```

### Пример выходных данных
```
Много
Мало
Много
Выход по желанию пользователя
```

(конкретная последовательность зависит от загаданного числа и выбранного диапазона)

### Критерии успешного выполнения
- используется цикл while с условием продолжения
- есть выход из цикла при угадывании числа и при вводе нуля
- для каждого ввода выводится корректное сообщение
- программа не уходит в бесконечный цикл

### Подсказки
- сохрани загаданное число в переменной перед циклом
- в цикле читай ввод пользователя и сначала проверяй особый случай с нулем
- затем сравни введенное число с загаданным и выведи сообщение
- для завершения цикла используй break

### Частые ошибки и ограничения
- отсутствие изменения состояния внутри цикла. из за чего условие никогда не становится ложным
- сравнение строк вместо целых чисел
- забытый случай с нулем
- можно не генерировать случайное число. достаточно выбрать фиксированное значение внутри программы
