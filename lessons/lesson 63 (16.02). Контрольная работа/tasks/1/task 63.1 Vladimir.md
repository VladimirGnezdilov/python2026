# Практическое задание 63.1 (HTML)

## Задание
Сделай простую страницу «Моя визитка».

### Требования
- Страница должна быть в файле `index.html`.
- Используй базовую структуру HTML: `<!doctype html>`, `html`, `head`, `body`.
- В `head` укажи `meta charset="UTF-8"` и `title`.
- В `body` добавь:
  - Заголовок `h1` с твоим именем
  - Абзац `p` с 2–3 предложениями о тебе
  - Список `ul` из 3 пунктов (например: хобби, любимые технологии, цель)
  - Ссылку `a` на любой сайт (например, на поисковик)
  - Картинку `img` (можно любую картинку из интернета через ссылку)

### Что проверить перед сдачей
- Все теги закрыты.
- У картинки есть `alt`.
- У ссылки есть `href`.

### Стартовый шаблон (можно скопировать)
```html
<!doctype html>
<html lang="ru">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Моя визитка</title>
  </head>
  <body>
    <h1>Имя Фамилия</h1>

    <p>Короткий текст о себе.</p>

    <h2>Мои факты</h2>
    <ul>
      <li>Факт 1</li>
      <li>Факт 2</li>
      <li>Факт 3</li>
    </ul>

    <p>
      Ссылка: <a href="https://example.com" target="_blank">Открыть сайт</a>
    </p>

    <img
      src="https://via.placeholder.com/320x200"
      alt="Пример картинки"
      width="320"
      height="200"
    />
  </body>
</html>
```
