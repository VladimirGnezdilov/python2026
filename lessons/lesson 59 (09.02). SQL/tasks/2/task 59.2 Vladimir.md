# Уровень 1: Базовый

## Задача

Создайте таблицу в базе данных SQLite, вставьте несколько записей и выполните запрос для получения всех записей из таблицы.

## Условия

- Нужно создать таблицу для хранения данных о студентах. Для каждого студента указываем:
  - уникальный идентификатор (id)
  - имя (name)
  - возраст (age)
- Вставьте несколько записей с разными значениями.
- Выполните запрос, чтобы получить все записи из таблицы и отобразить их.

## Пример псевдокода

```

// Добавить третьего студента (id=3, name="Петр", age=19)
// Получить все записи из таблицы
// Вывести результаты
// Закрыть соединение
```

# Уровень 2: Средний

## Задача

Создайте таблицу в базе данных SQLite, вставьте несколько записей, обновите одну из них и выполните запрос для получения всех записей с изменениями.

## Условия

- Создайте таблицу для хранения данных о сотрудниках:
  - уникальный идентификатор
  - имя
  - должность
  - зарплата
- Вставьте информацию о нескольких сотрудниках.
- Измените зарплату одного из сотрудников.
- Выполните запрос, чтобы получить все записи и увидеть изменения.

## Пример псевдокода

```
// Подключиться к базе данных
// Создать таблицу сотрудников с полями id, name, position, salary
// Добавить первого сотрудника (id=1, name="Анна", position="Разработчик", salary=50000)
// Добавить второго сотрудника (id=2, name="Олег", position="Менеджер", salary=60000)

```

# Уровень 3: Продвинутый

## Задача

Создайте таблицу в базе данных SQLite, вставьте несколько записей, выполните сначала удаление одной из них, а затем запрос для получения всех оставшихся записей.

## Условия

- Создайте таблицу для хранения данных о товарах:
  - уникальный идентификатор
  - название
  - цена
  - количество в наличии
- Вставьте несколько товаров.
- Удалите один товар из таблицы по идентификатору.
- Выполните запрос, чтобы получить все оставшиеся товары.

## Пример псевдокода

```
// Подключиться к базе данных
// Создать таблицу товаров с полями id, name, price, quantity
// Добавить первый товар (id=1, name="Ноутбук", price=45000, quantity=10)
// Добавить второй товар (id=2, name="Мышь", price=1500, quantity=50)
// Добавить третий товар (id=3, name="Клавиатура", price=3000, quantity=25)
// Добавить четвертый товар (id=4, name="Монитор", price=12000, quantity=15)
// Удалить товар с id=3 из таблицы
// Получить все оставшиеся записи из таблицы
// Вывести результаты (товары 1, 2, 4)
// Закрыть соединение
```
