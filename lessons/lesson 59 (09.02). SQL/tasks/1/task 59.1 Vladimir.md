# Задание 59.1: SQL-запросы к таблице Titanic

## Уровень 1: Базовый

### Задача:
Напишите SQL-запросы для извлечения следующих данных из таблицы `titanic`, которую мы использовали на занятии:

1. Извлечь все данные о пассажирах без дополнительных условий
2. Извлечь информацию об именах пассажиров старше 30 лет
3. Получить список имен всех женщин, которые путешествовали первым классом

## Пример псевдокода

```
// Задача 1: Извлечь все данные о пассажирах
// ВЫБРАТЬ все колонки
// ИЗ таблицы titanic

// Задача 2: Извлечь имена пассажиров старше 30
// ВЫБРАТЬ колонку с именами
// ИЗ таблицы titanic
// ГДЕ возраст больше 30

// Задача 3: Получить имена женщин первого класса
// ВЫБРАТЬ колонку с именами
// ИЗ таблицы titanic
// ГДЕ пол равно 'женский' И класс равен 1
```

## Уровень 2: Средний

### Задача:
Напишите SQL-запросы к таблице `titanic`, используя фильтрацию, сортировку и работу с несколькими столбцами:

1. Выведите имена и возраст выживших пассажиров, используя сортировку по возрасту
2. Выведите имена пассажиров, которые путешествовали одни (без братьев, сестёр, супруга или супруги, родителей или детей)
3. Выведите имена и классы мест тех пассажиров, стоимость билета которых превышала 100

## Пример псевдокода

```
// Задача 1: Имена и возраст выживших с сортировкой
// ВЫБРАТЬ колонки имя и возраст
// ИЗ таблицы titanic
// ГДЕ статус выживания равен 'выжил'
// СОРТИРОВАТЬ по возрасту по возрастанию

// Задача 2: Имена пассажиров без родственников
// ВЫБРАТЬ колонку с именами
// ИЗ таблицы titanic
// ГДЕ количество братьев/сестер равно 0
//   И количество супругов равно 0
//   И количество родителей равно 0
//   И количество детей равно 0

// Задача 3: Имена и классы пассажиров с дорогими билетами
// ВЫБРАТЬ колонки имя и класс
// ИЗ таблицы titanic
// ГДЕ стоимость билета больше 100
```

## Уровень 3: Продвинутый

### Задача:
Напишите SQL-запросы по базе данных `titanic`, используя сложные условия фильтрации и отрицания:

1. Выведите список пассажиров (имя, класс, возраст), которые путешествовали не первым классом и были старше 18 лет
2. Выведите список пассажиров, которые не выжили и у которых не было родственников на борту
3. Выведите имена, стоимость билетов и класс мест тех пассажиров, которые заплатили за билет меньше 10, но путешествовали не третьим классом

## Пример псевдокода

```
// Задача 1: Пассажиры не первого класса старше 18
// ВЫБРАТЬ колонки имя, класс, возраст
// ИЗ таблицы titanic
// ГДЕ класс не равен 1 И возраст больше 18

// Задача 2: Не выжившие без родственников
// ВЫБРАТЬ все колонки
// ИЗ таблицы titanic
// ГДЕ статус выживания равен 'не выжил'
//   И общее количество родственников равно 0

// Задача 3: Дешевые билеты не в третьем классе
// ВЫБРАТЬ колонки имя, стоимость билета, класс
// ИЗ таблицы titanic
// ГДЕ стоимость билета меньше 10 И класс не равен 3
```
