# Домашнее задание 59.2: Создание и управление таблицами базы данных

## Описание задачи

Создайте структуру базы данных для хранения информации о книгах и авторах, используя только запросы для создания таблиц и удаления данных. После удаления данных таблицы должны сохраниться. Все записи должны быть добавлены и удалены с помощью SQL-запросов без удаления самих таблиц.

## Указания

### Шаг 1: Создание базы данных

Создайте базу данных с именем `BookStore` (в SQLite база данных будет создаваться автоматически при подключении).

### Шаг 2: Создание таблицы Authors

Создайте таблицу `Authors` со следующими полями:

- `AuthorID` — уникальный идентификатор автора (тип данных INTEGER)
- `FirstName` — имя автора (тип данных TEXT)
- `LastName` — фамилия автора (тип данных TEXT)

### Шаг 3: Создание таблицы Books

Создайте таблицу `Books` со следующими полями:

- `BookID` — уникальный идентификатор книги (тип данных INTEGER)
- `Title` — название книги (тип данных TEXT)
- `AuthorID` — идентификатор автора, ссылающийся на таблицу Authors (тип данных INTEGER)
- `Price` — цена книги (тип данных REAL)

### Шаг 4: Очистка таблиц от данных

Очистите таблицы от всех записей, не удаляя сами таблицы. Для этого используйте команду `DELETE`.

### Шаг 5: Проверка сохранения таблиц

Сохраните таблицы до и после удаления данных. Таблицы должны быть сохранены: после выполнения запроса на удаление данных структура таблиц должна оставаться в базе данных.
