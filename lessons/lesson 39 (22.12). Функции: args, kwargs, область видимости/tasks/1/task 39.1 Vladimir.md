# Практическое задание 1

## Задания

### Описание задачи
Отработайте навыки работы с `*args`, средним арифметическим и округлением. Нужно написать функцию анализа результатов проверочной, которая принимает произвольное количество баллов и возвращает кортеж абсолютных отклонений каждого балла от среднего. Контекст из урока: повторяем, как собирать позиционные аргументы и грамотно обрабатывать пустой ввод, чтобы не получать деление на ноль. Ожидаемый результат — кортеж из чисел с точностью до одной десятой. Типичные ошибки: забыть проверить отсутствие аргументов, вернуть список вместо кортежа, округлять только итог а не каждый элемент.

### Пример входных и выходных данных

```python
print(mean_abs_deviation(3, 5, 7, 9))
# input: 3, 5, 7, 9
# steps: avg = 6.0, deviations = (3.0, 1.0, 1.0, 3.0)
# output: (3.0, 1.0, 1.0, 3.0)

existing_scores = [10, 8, 6]
print(mean_abs_deviation(*existing_scores))
# демонстрирует распаковку списка
```

- Подсказки: используйте `len(values)` и `sum(values)`, метод кортежей не нужен, но итог можно получить через `tuple(list)`. Проверку пустого ввода делайте до расчета среднего.

### Критерии проверки и ограничения
1. Вызов без аргументов обязан поднимать `ValueError` с осмысленным текстом.
2. Каждое отклонение округляется до одной десятой, используйте `round(value, 1)`.
3. Возвращаемый тип — кортеж, длина равна количеству входных значений.
4. Допускаются только числовые аргументы. По желанию можете добавить явную проверку `isinstance(value, (int, float))`.

### Решение задачи (псевдокод)
1. Если количество переданных значений равно нулю, остановить выполнение с сообщением об ошибке.
2. Вычислить среднее значение: сумма всех чисел, деленная на их количество.
3. Для каждого числа вычислить абсолютное отклонение от среднего и округлить до одной десятой.
4. Сохранить отклонения в неизменяемую последовательность.
5. Вернуть последовательность.
