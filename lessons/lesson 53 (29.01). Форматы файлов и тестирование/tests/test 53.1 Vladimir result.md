# Вопрос 1. Тема: Форматы JSON и CSV
  
**1. Что из перечисленного НЕ является допустимым типом данных в JSON?**
  
Варианты ответов:
1) Словарь
2) Строка
3) ✅ Функция
  
### ИИ анализ
Правильный ответ — «Функция». JSON — это текстовый формат для обмена данными, и он поддерживает ограниченный набор типов: строки, числа, булевы значения, null, массивы и объекты. Функции, классы или другие исполняемые элементы не являются допустимыми типами данных в JSON, так как формат предназначен только для данных, а не для кода.  

# Вопрос 2. Тема: Форматы JSON и CSV
  
**2. Какой параметр позволяет улучшить читаемость JSON-файлов при записи?**
  
Варианты ответов:
1) formatter
2) newline
3) ✅ indent
  
### ИИ анализ
Правильный ответ — `indent`. При записи JSON-файла с помощью функции `json.dump()` или `json.dumps()`, параметр `indent` определяет количество пробелов для отступов, что форматирует вывод с вложенными структурами, делая его удобочитаемым для человека. Другие параметры, такие как `newline`, управляют символами перевода строки, но не улучшают визуальную структуру.  

# Вопрос 3. Тема: Форматы JSON и CSV
  
**3. Какой метод используется для записи данных в JSON-файл?**
  
Варианты ответов:
1) ✅ json.dump()
2) json.save()
3) json.write()
  
### ИИ анализ
Правильный ответ — `json.dump()`. Этот метод принимает объект Python и файловый объект, сериализуя данные в JSON-формат и записывая их в файл. Методы `json.save()` и `json.write()` не существуют в стандартной библиотеке Python для работы с JSON.  

# Вопрос 4. Тема: Форматы JSON и CSV
  
**4. Какой метод используется для записи данных в CSV-файл?**
  
Варианты ответов:
1) csv.read()
2) ✅ csv.writerows()
3) csv.dump()
  
### ИИ анализ
Правильный ответ — `csv.writerows()`. Этот метод объекта writer принимает список строк (каждая строка — это список значений) и записывает их все в CSV-файл за одну операцию. Метод `csv.read()` используется для чтения, а `csv.dump()` не является стандартным методом модуля csv.  

# Вопрос 5. Тема: Форматы JSON и CSV
  
**5. Какой из следующих форматов поддерживает вложенные структуры данных?**
  
Варианты ответов:
1) ❌ Оба
2) JSON
3) CSV
  
### ИИ анализ
Правильный ответ — «JSON». Формат JSON изначально поддерживает вложенные объекты и массивы, что позволяет создавать сложные древовидные структуры данных. CSV — это табличный формат, где данные представлены в виде плоских строк и столбцов; он не поддерживает вложенность напрямую, хотя её можно эмулировать с помощью дополнительного парсинга.  

# Вопрос 6. Тема: Форматы JSON и CSV
  
**6. Выберите верные утверждения о JSON (выбор нескольких вариантов):**
  
Варианты ответов:
1) JSON можно использовать только в Python
2) ❌ JSON поддерживает числовые типы данных
3) JSON не поддерживает тип данных null
4) JSON поддерживает вложенные структуры
  
### ИИ анализ
Правильный ответ — «JSON поддерживает числовые типы данных» и «JSON поддерживает вложенные структуры». JSON — это независимый от языка формат, используемый во многих технологиях. Он поддерживает числа (целые и с плавающей точкой), строки, булевы значения, null, массивы и объекты, причём объекты и массивы могут быть вложенными. Утверждение о том, что JSON можно использовать только в Python, неверно.
