# Описание задачи

Создать функцию, которая принимает на вход строку и возвращает количество гласных букв в ней. Для этой функции нужно написать модульное тестирование с использованием модуля `unittest`.

## Указания

Реализуйте функцию `count_vowels`, которая принимает строку и подсчитывает количество гласных букв (a, e, i, o, u) в этой строке. Функция должна быть нечувствительна к регистру.

Напишите модульное тестирование для проверки:

- корректности подсчёта гласных в строках разного типа;
- обработки пустой строки;
- работы функции при вводе строки без гласных.

## Ожидаемый результат

- Функция `count_vowels` должна правильно подсчитывать количество гласных в строках в независимости от регистра символов.
- Тесты должны проверять как положительные, так и отрицательные случаи.
