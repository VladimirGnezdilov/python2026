### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая нормализует строку с названием компании.

Требования:
- убрать пробелы по краям строки
- заменить несколько пробелов внутри на один
- привести каждое слово к виду с заглавной первой буквой
- вывести количество слов и длину строки

### Пример входных данных
```
  VLADIMIR   soft   solutions 
```

### Пример выходных данных
```
Название: Vladimir Soft Solutions
Слов: 3
Длина: 24
```

### Критерии успешного выполнения
- строка очищена от лишних пробелов
- каждое слово приведено к нужному регистру
- длина и количество слов посчитаны корректно

### Подсказки
- используй `strip`, `split`, `' '.join(...)`, `title()` или свой вариант

### Частые ошибки и ограничения
- не модифицируй строку по индексу
- учитывай пустой ввод
