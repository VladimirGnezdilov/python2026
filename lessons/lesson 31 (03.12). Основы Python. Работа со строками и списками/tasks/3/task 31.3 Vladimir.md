### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая формирует перечень модулей проекта.

Требования:
- прочитать строку с названиями модулей, разделенными запятой
- удалить пробелы по краям каждого названия
- привести все названия к нижнему регистру
- отсортировать по алфавиту и вывести по одному модулю в строке с нумерацией

### Пример входных данных
```
  Core, Utils, api, Database, core 
```

### Пример выходных данных
```
1. api
2. core
3. database
4. utils
```

### Критерии успешного выполнения
- строка корректно разбита на элементы
- пробелы удалены
- список отсортирован и выведен с нумерацией

### Подсказки
- используй `split(',')`, `strip()`, `lower()`, `sorted`

### Частые ошибки и ограничения
- можешь убрать дубликаты перед сортировкой
- не используй внешние библиотеки
