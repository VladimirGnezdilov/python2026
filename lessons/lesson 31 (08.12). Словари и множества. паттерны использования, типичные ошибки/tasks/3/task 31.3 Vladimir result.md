#### 1. Формулировка задания
**Цель:**  
Сравнить два множества номеров грузовиков (утренние и вечерние рейсы), найти:
- Общие номера (присутствуют в обоих множествах).
- Номера, свободные вечером (присутствуют в вечернем множестве, но отсутствуют в утреннем).

**Входные данные:**  
Два множества строк (например, `set_a = {'TR-05', 'TR-01', 'TR-08'}`, `set_b = {'TR-05', 'TR-09', 'TR-10'}`).

**Ожидаемый вывод:**  
Два множества с пояснениями:
```text
Общие машины: {'TR-05'}
Свободны вечером: {'TR-09', 'TR-10'}
```

**Ключевые ограничения:**
- Использовать операции над множествами (`&`, `-`, `|`).
- Не преобразовывать множества в списки без необходимости.
- Вывод должен содержать осмысленные подписи.

---

#### 2. Результаты проверки
**Файл:** `task 31.3`  
**Код студента:**
```python
set_a = {'TR-05', 'TR-01', 'TR-08'}
set_b = {'TR-05', 'TR-09', 'TR-10'}
print(f"Общие машины:{set_a & set_b},\n Свободны вечером:{set_b - set_a}")
```

**Тестирование:**
- Запущен код с примером входных данных.
- **Результат вывода:**
  ```text
  Общие машины:{'TR-05'},
   Свободны вечером:{'TR-10', 'TR-09'}
  ```
- **Ошибки воспроизведения:**  
  Нет ошибок в логике, но есть незначительные отклонения в оформлении вывода (см. раздел 4).
