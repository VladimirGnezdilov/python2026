# Задача 45.1: Создание класса Book

Создай класс `Book` (книга), который описывает простой объект с характеристиками и действиями.

## Требования

1. **Конструктор `__init__`** должен принимать и сохранять в полях объекта:
   - `title` (название книги, строка)
   - `author` (автор книги, строка)
   - `pages` (количество страниц, число)

2. Добавь методы:
   - `open()`  
     Должен выводить фразу вида:  
     `Книга "<title>" открыта на первой странице.`
   - `read()`  
     Должен выводить фразу вида:  
     `Читаем книгу "<title>" автора {author}.`
   - `close()`  
     Должен выводить фразу вида:  
     `Книга "<title>" закрыта.`
   - `info()`  
     Должен выводить фразу вида:  
     `"{title}" - {author}, {pages} стр.`

3. В конце программы:
   - создай объект книги (например, название `"1984"`, автор `"Джордж Оруэлл"`, страниц `328`)
   - вызови методы `info()`, `open()`, `read()`, `close()` в таком порядке

## Пример вывода
```
"1984" - Джордж Оруэлл, 328 стр.
Книга "1984" открыта на первой странице.
Читаем книгу "1984" автора Джордж Оруэлл.
Книга "1984" закрыта.
```
