# Домашнее задание 61.2 · Сложные запросы

## Описание задачи
Создайте базу данных, в которой таблица хранит сведения о студентах и курсах. С помощью вложенного запроса найдите студентов, записанных на курс по программированию.

## Требования к структуре данных
Таблица **Students** должна содержать поля:
- `id` — уникальный идентификатор студента, первичный ключ.
- `name` — имя студента.
- `birth_date` — дата рождения.
- `email` — уникальный адрес электронной почты.
- `course_name` — название курса.
- `enrollment_date` — дата зачисления на курс.

## Рекомендуемая последовательность действий
1. Создайте базу данных (например, в DB Browser for SQLite).
2. Опишите таблицу **Students** с указанными столбцами и ограничениями.
3. Добавьте несколько записей со студентами, включая те, кто посещает курсы по программированию и по другим дисциплинам.
4. Сформулируйте вложенный запрос, который возвращает имена студентов, записанных на курс «Программирование».
5. Убедитесь, что запрос корректно фильтрует студентов по нужному курсу и возвращает ожидаемый результат.

## Ожидаемый результат
- Таблица **Students** создана и заполнена тестовыми данными.
- Вложенный запрос корректно возвращает студентов, записанных на указанный курс.
- Структура данных и запрос проходят проверку без ошибок.

## Критерии оценивания
- 4 балла — создана база данных и таблица.
- 4 балла — добавлены данные о студентах и курсах.
- 4 балла — выполнен вложенный запрос для получения студентов, записанных на определённый курс.

**Максимальный балл: 12.**

## Псевдокод решения
```
1. Инициализировать базу данных с одной таблицей Students.
2. Описать столбцы: идентификатор, имя, дата рождения, email, название курса, дата зачисления.
3. Внести несколько строк с разными курсами, включая программирование.
4. Сформировать вложенный запрос:
   a. Внутренний блок выбирает контакты студентов по названию курса «Программирование».
   b. Внешний блок выбирает имена тех студентов, чьи контакты присутствуют в результате внутреннего блока.
5. Проверить, что вывод содержит только студентов нужного курса.
```
