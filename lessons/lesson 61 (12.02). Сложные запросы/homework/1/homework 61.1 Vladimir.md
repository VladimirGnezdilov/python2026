# Домашнее задание 61.1 · Сложные запросы

## Описание задачи
Создайте базу данных для хранения информации о студентах и курсах. Сформируйте таблицы, добавьте несколько записей, получите список студентов с их курсами, затем очистите таблицы от данных (не удаляя структуру).

## Требования
1. Создать базу данных с именем **University**.
2. Создать таблицу **Students** со столбцами:
   - `StudentID` — уникальный идентификатор (INTEGER).
   - `FirstName` — имя (TEXT).
   - `LastName` — фамилия (TEXT).
3. Создать таблицу **Courses** со столбцами:
   - `CourseID` — уникальный идентификатор (INTEGER).
   - `CourseName` — название (TEXT).
   - `Credits` — количество кредитов (INTEGER).
4. Создать таблицу **Enrollments** для учета записей студентов на курсы со столбцами:
   - `StudentID` — ссылка на `Students.StudentID` (INTEGER).
   - `CourseID` — ссылка на `Courses.CourseID` (INTEGER).
5. Добавить несколько записей в каждую таблицу.
6. Выполнить запрос, который выводит студентов и соответствующие им курсы.
7. Очистить таблицы от данных, сохранив их структуру.

## Рекомендуемая последовательность действий
1. Создайте базу данных **University** (например, в DB Browser for SQLite).
2. Сформируйте таблицу **Students** с указанными столбцами.
3. Сформируйте таблицу **Courses** с указанными столбцами.
4. Сформируйте таблицу **Enrollments** с внешними ключами на ранее созданные таблицы.
5. Добавьте тестовые записи в **Students**, **Courses** и **Enrollments** (не менее по три строки).
6. Выполните запрос, который выводит фамилию, имя студента и название курса, объединив таблицы **Students**, **Enrollments** и **Courses**.
7. Удалите все строки из таблиц **Enrollments**, **Students** и **Courses**, не затрагивая сами таблицы.

## Ожидаемый результат
- База данных **University** создана и содержит таблицы **Students**, **Courses** и **Enrollments**.
- В таблицы добавлены несколько записей о студентах, курсах и их зачислениях.
- Выполнение запроса возвращает список студентов и курсов, на которые они записаны.
- Таблицы очищены от данных, но остаются в базе.
